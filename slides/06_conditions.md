# Branches
## Asking questions of your code

::left::



### Branch Example

```js {*|3|*}{lines:true,startLine:1}
let x = 10;
if (x > 5) {
  rect(0, 0, 50, 50);
}
```


