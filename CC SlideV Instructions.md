# Welcome to Creative Coding Slides

### Clone a Copy Repo from:
https://github.com/montaynej/CreativeCoding.git using git clone

- git clone https://github.com/montaynej/CreativeCoding.git
- cd to the new Folder

if on new machine
- rm -rf node_modules 
- rm package-lock.json
- npm install -g pnpm 
 
Install Node Modules which are included in GIT Ignore
- pnpm install    

Run the dev environment
- pnpm dev   

- visit <http://localhost:3030>

To Back Up
- git add .
- git commit -m "dsdsd"
- git push

Edit the [slides.md](./slides.md) to see the changes.

Learn more about Slidev at the [documentation](https://sli.dev/).
